1. Les gentilés

Définition : Un gentilé désigne les habitants d’un pays, d’une ville ou d’une région.

Formation : Généralement on ajoute un suffixe au nom du lieu e:

  • -ain / -aine : Maroc : Marocain (masculin) / Marocaine(féminin)
  • -ien / -ienne : Italie : Italien (masculin) / Italienne (féminin)
  • -ais / -aise : France : Français (masculin) / Française (féminin)
  • -in / -ine : Argentine : Argentin (masculin) / Argentine (féminin
  • -ois / -oise : Chine : Chinois (masculin) / Chinoise (féminin)
  • -an / -ane : Amérique : Américain (masculin) / Américaine (féminin)
  • -ol / -ole : Espagne : Espagnol (masculin) / Espagnole (féminin)

Applications :

  • Les habitants du Maroc sont des … Marocains.
  • Les habitants de l’Espagne sont des … Espagnols.

2. Le dictionnaire

Rôle : Comprendre les différents sens d'un mot, sa prononciation et sa classe grammaticale(nom (n), verbe (v), adjectif (adj) adverbe (adv) ... ).

Un article contient : mot - nature - définition - exemple - synonymes.

Exemple :

Extraordinaire (adj.) : qui sort de l’ordinaire.
Ex : Ce travail est extraordinaire.

Applications :

  • Cherche le mot utile et note sa définition.
  • Trouve dans le dictionnaire un synonyme de important.

3. Les synonymes

Définition : Les synonymes sont des mots qui ont le même sens ou presque le meme sens.

Exemples :

  • ami : camarade
  • content : joyeux
  • rapide : vite
  • lentement : doucement
  • important : essentiel
  • difficile : dure
  • beau : magnifique
  • travailler : exercer
  • parler : discuter
  • aimer : adorer
  • petit : miniscule
  • grand : large
  • facile : aiser
  • commencer : débuter
  • finir : terminer
  • regarder : voir
  • aider : assisster
  • lieu : endroit ...

Applications :

  • Donne un synonyme de commencer .
  • Remplace : Il est triste → Il est …

4. Lexique de la citoyenneté

Définition : La citoyenneté, c’est respecter les droits et les devoirs.

Mots clés : citoyen - comportement - droits - devoirs - respect - solidarité liberté - responsabilité - égalité - fraternité - paix - civisme - justice - valeurs - principe - éthique - politesse - respect mutuel - altruisme - empathie - droit à l'éducation - dialogue - médiation - conseiller - donner son avis - tavailler en équipe - coopération - société - identité - inclusion - diversité - cohésion sociale - éco-geste - propreté - recyclage - économiser d'eau - développement durable - économiser d'énergie - prtéger l'environnement - vote - élection - démocratie - engagement - débat - opinion - liberté d'expression - égalité des chances -acceptation - non-descrimination - acceptation des différences -communauté - patrie ...

Exemples :

  • Le droit d’aller à l’école.
  • Je trie mes déchets.
  • Je respecte les règles.

Applications :

  • Recycler → devoir, responsabilité, économiser d'énergie ...
  • Être soigné → droit,.
  • Utiliser des sacs en papier → éco-geste, propreté ...

5. Les antonymes

Définition : Les antonymes sont des mots de sens opposé.

Exemples :

  • grand ≠ petit
  • entrer ≠ sortir
  • heureux ≠ malheureux
  • vivre ≠ mourir
  • haut ≠ bas
  • riche ≠ pauvre
  • rapide ≠ lent
  • jeune ≠ vieux
  • propre ≠ sale
  • aimer ≠ détester
  • commencer ≠ finir
  • donner ≠ recevoir
  • acheter ≠ vendre
  • gagner ≠ perdre
  • ouvrir ≠ fermer
  • avancer ≠ reculer
  • travailler ≠ se reposer
  • se lever ≠ s'endormir
  • protéger ≠ attaquer
  • mémoriser ≠ oublier
  • jour ≠ nuit
  • amour ≠ haine
  • paix ≠ guerre
  • succès ≠ echec
  • silence ≠ bruit
  • vérité ≠ mensonge
  • ami ≠ ennemi
  • victoire ≠ défaite
  • présence ≠ absence
  • toujours ≠ jamais
  • souvent ≠ rarement
  • tôt ≠ tard
  • tout ≠ rien
  • beaucoup ≠ peu
  • avant ≠ après
  • dadans ≠ dehors
  • devant ≠ derrière

Règle importante à retenir : Les antonymes appartiennent à la même catégorie grammaticale.

Applications :

  • Antonyme de propre : …
  • Antonyme de rapide : …

6. Lexique de la technologie

Définition : Vocabulaire lié à l’informatique, Internet et aux médias numériques.

Mots :

  • GPS : système global de localisation
  • Logiciel : programme informatique
  • Scanner : transformer en fichier numérique
  • IPTV : télévision par Internet

Applications :

  • Je clique sur un … lien.
  • J’envoie un … e-mail.
